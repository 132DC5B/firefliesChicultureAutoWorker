# 篇篇流螢自動應答器 (Fireflies Chiculture Auto Worker)
夫此器者，專為「篇篇流螢（Fireflies Chiculture）」平臺所製。其能自動檢索作業，擷取正解，並以滿分之姿代為提交。

## I. 功能概覽
* **自動檢索**：依據 `date.txt` 所列之日期，自動搜尋對應之作業。
* **答案擷取**：利用管理員權限（需備 Cookie），逕向伺服器探取標準答案。
* **滿分提交**：自動填入正解，類比學生作答並提交。
* **批量處理**：可一次性處理多日之作業。
* **宏定義**：可以決定是否啟用的功能。

## II. 所需之物
欲行此器，需備以下環境與套件：
1.  **作業系統**：Windows (x64/x86)。
2.  **編譯工具**：Visual Studio 2022 (MSVC v143) 或相容之 C++ 編譯器。
3.  **依賴庫**：
    * `libcurl`：用於網路傳輸。
    * `cJSON`：用於解析 JSON 數據（本案已內含 `cJSON.c` 與 `cJSON.h`）。

## III. 編譯與配置

### i. 編譯之法
本案附有 Visual Studio 專案檔（`.sln`），可直接開啟並生成之。
若編譯時報錯，務請檢查 `libcurl` 之路徑設定是否正確。

### ii. 配置檔案
於程式運行目錄下，需自行建立或確認以下檔案存在：

#### 1. `date.txt` (日期清單)
列出欲自動完成之作業日期，一行一期，格式如 `YYYY-MM-DD`。
**範例：**
```text
2025-12-22
2025-12-19
2025-10-17
```

#### 2.`acc.txt`(提供的帳戶和密碼)
欲為己做，則提供汝之帳密。
**範例**
```text
s000000@xxxxxxx.xxx
Xx00000000
```
### iii. 開始

#### 1. 確保`acc.txt`在同一目錄下，執行`getAccessID.exe`。
此將獲得汝帳密之cookies，生成`cookies_student.txt`。

#### 2. 提供`cookies_admin.txt`，安全起，此倉庫不提供。

#### 3. 執行`firefliesChicultureAutoWorker.exe`，並等待結束。

### iv. 結束，任務已完成。
